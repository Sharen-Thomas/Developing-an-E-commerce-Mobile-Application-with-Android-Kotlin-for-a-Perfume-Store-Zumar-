# Developing-an-E-commerce-Mobile-Application-with-Android-Kotlin-for-a-Perfume-Store-Zumar-

## OVERVIEW

Zumar is a fully functional Android e-commerce app built with Kotlin for a perfume store. Features secure authentication, product discovery, cart management, multi-currency support, and real-time order tracking, delivering a seamless mobile shopping experience.

## WHAT IT DOES

### AUTHENTICATION
- Secure signup/login with Firebase Auth, password strength validation

### PRODUCT DISCOVERY
- Gender filters (Men/Women/Unisex), search, image sliders, fragrance notes

### SHOPPING CART
- Add/remove items, quantity control, coupon codes (SAVE10, WELCOME20, PROMO50)

### PAYMENT
- Cash on Delivery & Card Payment with input validation

### ADDRESS MANAGEMENT 
- Add/edit/delete addresses, GPS location fetch

### ORDER TRACKING 
- Dashboard with status (Processing/Shipped/Delivered/Cancelled)

### INTERNATIONALISM 
- Multi-currency (UAE, USA, UK, India, Canada), Language options, Dark mode

## KEY FEATURES

### AUTHENTICATION 
- Email/password signup with phone verification
- Session management (auto-login persistence)
- Real-time password strength indicator

### SHOPPING EXPERIENCE 
- Horizontal category browsing with DiffUtil optimization
- Dynamic delivery fee calculation based on cart total
- Real-time discount application with coupon codes

### TECHNICAL HIGHLIGHTS 
- MVVM Architecture + Repository Pattern
- Firebase Firestore for real-time data sync
- Navigation Component for seamless fragment transitions
- Custom "No Internet" dialog for offline handling

## TECH STACK 

- Language: Android Kotlin
- Architecture: MVVM + Repositary
- Backend: Firebase (Auth, Firestore)
- Image Loading: Glide
- Navigation: Android Navigation Component
- User Interface (UI): Material Design, Lucide Icons
